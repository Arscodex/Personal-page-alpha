Front End & UI/UX

Ball Jump

This is a simple puzzle game that uses SVG generated on the fly. There are no loaded assets. The objective is to eliminate balls on the game board by jumping over them with another ball. It was a big exercise in functional programming, and I had to think of every step of an action as a function.

Carousel Slider

A basic slideshow built for Wordpress. I set the condition that I would accomplish this without using any libraries such as jQuery. It helped me learn how much goes into developing the shortcuts we take for granted. The images are post thumbnails generated for the page through PHP and animated with Javascript and CSS.
